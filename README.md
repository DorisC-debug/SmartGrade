# SmartGrade

SmartGrade is an interactive web platform that functions as an **academic chatbot**. Its main goal is to help students create the **critical path of their curriculum**, making it easier to plan their university studies.

## Features

- Generation of curriculum critical paths.
- Chatbot-style interaction for academic inquiries.
- Web platform with a user-friendly and responsive interface.
- Client and server integration.

## Technologies

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, SQL Server, Express
- **Additional Tools:** OpenAI API

## Installation

1. Clone the repository:

```bash
git clone https://github.com/DorisC-debug/SmartGrade.git
````


2. Install dependencies for the client and server:
```bash
cd SmartGrade/client
npm install
cd ../server
npm install
```


3. Run the application:

```# On the server
npm start

# On the client
npm start
```
